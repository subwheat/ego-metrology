# experiments/

Contient le matériel de labo interne :

- `scripts/` : runners, calibrations, analyses ponctuelles
- `artifacts/data/` : corpus figés
- `artifacts/runs/` : JSONL de runs
- `artifacts/reports/` : résumés markdown/json
- `artifacts/calibration/` : résultats de calibration

Le noyau public du repo reste à la racine :
- `ego_metrology/`
- `tests/`
- `server.py`
- `README.md`
- `pyproject.toml`
- `.github/workflows/ci.yml`
